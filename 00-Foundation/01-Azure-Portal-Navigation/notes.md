# Azure Portal Shortcuts & Tricks

## Keyboard Shortcuts
1. **?** → Open the keyboard shortcuts overlay.
2. **g + r** → Navigate directly to **Resource Groups**.
3. **g + s** → Navigate to **Subscriptions**.
4. **g + a** → Navigate to **All Resources**.
5. **g + d** → Return to the **Dashboard**.
6. **g + c** → Open **Cloud Shell** instantly.
7. **g + n** → Create a new resource.
8. **/ (forward slash)** → Jump to the **global search bar**.
9. **Shift + Alt + D** → Pin the current blade to the dashboard.
10. **Ctrl + /** → Toggle the shortcuts overlay (alternative to `?`).

---

## Portal Tricks
- **Pinning Resources:** Any blade can be pinned to the dashboard for quick access.  
- **Dark Mode:** Switch themes under *Portal Settings → Appearance*.  
- **Resource Visualizer:** Enable preview to see resource dependencies graphically.  
- **Cloud Shell Persistence:** First launch creates a storage account automatically, ensuring scripts and files persist.  
- **Multiple Dashboards:** Create separate dashboards for projects or environments.  
- **Blade Navigation:** Use breadcrumbs at the top of blades to quickly jump back.  
- **Favorites Bar:** Customize the left-hand navigation bar with your most-used services.  
- **Search Filters:** Narrow global search results by subscription, resource type, or location.  
- **Activity Log:** Track every action in the portal — useful for auditing.  
- **Preview Features:** Enable experimental features under *Portal Settings → Preview Features*.

## Notes
These shortcuts and tricks significantly improve efficiency when working in the Azure Portal.  
They reduce navigation time, enhance visibility, and provide deeper insights into resource relationships.

# Azure Advisor – The Five Pillars

Azure Advisor is a free, personalized cloud consultant that analyzes your resource configuration and usage telemetry. It provides recommendations across five pillars:

1. **Cost**  
   Helps you optimize and reduce overall Azure spend. Examples: idle resources, right‑sizing, reserved instances.

2. **Security**  
   Alerts you to potential vulnerabilities and configuration weaknesses. Examples: MFA, network security group rules, encryption settings.

3. **Reliability**  
   Ensures your applications are highly available and resilient. Examples: backup configuration, availability zones, disaster recovery.

4. **Operational Excellence**  
   Improves process and workflow efficiency. Examples: diagnostics settings, agent upgrades, removing obsolete resources.

5. **Performance**  
   Helps you improve the speed and responsiveness of your applications. Examples: VM sizing, accelerated networking, managed disks.

## Key Features
- **Advisor Score**: A numeric score reflecting how well your environment aligns with best practices.
- **Quick Fix**: Some recommendations allow one‑click remediation (e.g., enabling soft delete).
- **Alerts**: Configure proactive notifications when new high‑impact recommendations are found.

## How We Used Advisor
- Reviewed recommendations under all five pillars and documented the top 3 each.
- Applied a Quick Fix (or described the process) to immediately improve security.
- Set up a security alert to be notified of future critical security findings.
- Exported the overall Advisor score for reporting.

Advisor is an essential part of continuous governance – it’s your free, always‑on cloud health check.
